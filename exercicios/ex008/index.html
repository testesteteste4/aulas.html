Principais formataçoes

negrito / destaque

Nesta frase temos um termo em negrito usando a "tag b"

usando apenas a tag "b" ela se forma uma frase (nao semântica): sem sentido, apenas forma

Nesta frase temos um termo em negrito usando a "tag strong"

usando apenas a tag "strong" ela se forma uma frase (semântica): que da sentido a frase

Nesta frase temos um termo em "italico usando a tag (i)" (nao semântica) sem sentido, apenas forma

Nesta frase temos um termo em "italico usando a tag (em)" (semântica) que da sentido a frase

"TEXTO PEQUENO"

nesta frase vamos ter um ''texto pequeno'' usando a tag "small"

"texto deletado"

podemos usar um texto excluido que possa ser lido, mas nao considerado usando a tag (del)

"Texto com linha abaixo (texto inserido)"

Podemos marcar um texto como inserido, com uma linha abaixo (semântica)
